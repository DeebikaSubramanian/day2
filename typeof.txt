Description about typeof() in javascript

1. typeOf(1):
	type of an integer 1 is number

2. typeOf(1,1)
	(1,1) is also belongs to type of number

3. typeOf('1,1')
	single quoted input belongs to type of string

4. typeOf(true)
	true and false are booleon type in javascript

5. typeOf(null)
	null belongs to an object

6. typeOf(undefined)
	undefined belongs to undefined data type

7. typeOf([])
	[] is an object type.

8. typeOf({})
	{} is an object 

9. typeOf(NaN)
	Nan is a number but returns Nan when there is no any number declared
